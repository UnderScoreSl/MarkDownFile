## windows

### 关闭触屏

fn + f6



### 分屏

win + p



### 切换多桌面

ctrl + win + d  :  新建虚拟桌面

ctrl + win + 左右键  :  切换桌面

win + tab :  查看所有桌面, 可以手动点任意的桌面

win + d  :  切换到当前桌面 (最小化所有应用) 



### 切换应用

alt + tab
